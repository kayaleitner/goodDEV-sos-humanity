# BlockWysiwyg

## Overview
`BlockWysiwyg` is a Flynt 2 component for displaying a title and WYSIWYG content in a structured grid layout. It supports customizable background and text colors via ACF fields.

## ACF Fields
### **Text Tab**
| Field Name    | Type    | Description             |
|--------------|--------|-------------------------|
| `blockTitle`  | Text   | H2 title for the block  |
| `contentHtml` | WYSIWYG | Main content area |

### **Options Tab**
| Field Name         | Type        | Description |
|-------------------|------------|-------------|
| `componentId`     | Text       | Unique component ID |
| `colorBackground` | Color Picker | Background color |
| `colorText`       | Color Picker | Text color |
